##物流管理系统API

* 备注：每个接口的url都去掉了服务器路径地址
一般服务器地址为：localhost:8080,本API只给与URI和查询字段
* 错误代码备注：查看错误代码文档
* 成功返回若无数据要求，一般返回状态码 1 ，信息为： SUCCESS
* 如果有返回信息需求，请明确指出字段格式

###公共接口API

* 接口名：登陆
* 使用介绍：用于用户进入系统的入口，验证为系统用户才能使用该系统功能
* 使用权限：财务管理、系统管理、用户
* uri:/public/login
* 参数
{
   account:用户账号
   password:用户加密密码
  
}
